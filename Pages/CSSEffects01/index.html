一些使用 CSS 制作的简单效果

背景设置 

            
<div class="example-container"></div>
            
            
.example-container{
    width: 20em;
    height: 20em;
    position: relative;
    /* 内部元素居中 */
    display: flex;
    justify-content: center;
    align-items: center;
    /* 渐变背景效果 */
    background-color: #454545;
    background-image: linear-gradient(
            135deg, rgba(0, 0, 0, 0), rgba(0, 0, 0, 0.2));
}

简单凹陷效果 

            
<div class="example-container">
    <div id="no01-circle">
        <span>测试</span>
    </div>
</div>
            
          
/* 内部 div 元素设置 */
#no01-circle {
    width: 10em;
    height: 10em;
    color: white;
    /* 文字居中 */
    display: flex;
    justify-content: center;
    align-items: center;
    /* 50% 圆角将 div 元素绘制成圆 */
    border-radius: 50%;
    /* 内部阴影设置达到凹陷效果 */
    box-shadow: 2em 0 3em rgba(0, 0, 0, 0.2) inset,
                -2em 0 3em rgba(255, 255, 255, 0.1) inset;
}

简单 3D 质感的按钮效果 

                
<div class="example-container" id="no02-container">
    <div class="no02-button">1</div>
    <div class="no02-button" id="no2-button-circle">2</div>
</div>
                
                
#no02-container {
    /* 设置内部元素的对其方式为间隔平均分布 */
    justify-content: space-around;
}
/* 按钮的外观 */
.no02-button {
    width: 3em;
    height: 3em;
    color: white;
    /* 文字居中 */
    display: flex;
    justify-content: center;
    align-items: center;
    /* 内部阴影绘制高光 */
    box-shadow: 0.05em 0 0.1em rgba(255, 255, 255, 0.2) inset;
}
.no02-button:before {
    content: "";
    width: 3em;
    height: 3em;
    position: absolute;
    /* 外部阴影绘制按钮阴影 */
    box-shadow: 0.1em 0.1em 0 0.1em black,
                0.3em 0 0.3em rgba(0, 0, 0, 0.5);
}
#no2-button-circle, #no2-button-circle:before {
    /* 通过圆角设置绘制圆形按钮 */
    border-radius: 50%;
}
