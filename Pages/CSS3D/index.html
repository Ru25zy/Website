使用纯 CSS 制作简单的 3D 图形
CSS3 提供的三维变换功能
CSS3 提供了 3D 转换功能，因此使用此功能可以创建一些简单的 3D 图形，以下是简单示例：
X 轴方向上的旋转 transform: rotateX(120deg);
这是一个 div 元素。
Y 轴方向上的旋转 transform: rotateY(120deg);
这是一个 div 元素。
Z 轴方向上的旋转 transform: rotateZ(120deg);
这是一个 div 元素。
向 X 轴正方向移动 50px transform: translateX(50px);
这是一个 div 元素。
向 Y 轴正方向移动 50px transform: translateY(50px);
这是一个 div 元素。
向 Z 轴正方向移动 50px transform: translateZ(50px);
这是一个 div 元素。
简单的纯 CSS 正方体演示动画
知道了以上变换之后就能够创建一个正方体了：
  1. 创建一个 div 元素作为正方体的容器。
  2. 在容器 div 元素中创建六个边长相等的 div 元素做为正方体的六个面。
  3. 将六个面旋转、移动到相应的位置。
  4. 现在只要对容器进行变换就能让正方体进行相应的变化。

纯 CSS 正方体示例代码
以下是一个正方体的代码示例，右侧是显示效果： 
                
/* 容器 */
#code-example-cube{
    width: 200px;
    height: 200px;
    position: relative;
    perspective: none;
    /* 指定变换方式为 3D */
    transform-style: preserve-3d;
    transform:rotateX(-30deg) rotateY(30deg);
}
/* 确定六个面的大小和样式 */
#code-example-cube div{
    width: 200px;
    height: 200px;
    border: 10px solid rgba(255,0,0,0.8);
    background: rgba(255,0,0,0.3);
    position: absolute;
}
/* 移动和旋转每个面组成一个正方体 */
#code-example-cube-left{
    transform: translateX(-100px) rotateY(90deg);
}
#code-example-cube-Right{
     transform: translateX(100px) rotateY(-90deg);
}
#code-example-cube-top{
    transform: translateY(-100px) rotateX(90deg);
}
#code-example-cube-bottom{
    transform: translateY(100px) rotateX(-90deg);
}
#code-example-cube-front{
    transform: translateZ(100px);
}
#code-example-cube-back{
    transform: translateZ(-100px);
}
